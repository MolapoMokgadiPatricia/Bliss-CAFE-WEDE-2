This is a simple static website built using HTML, CSS and Java Script.
It demonstrates basic web structure, styling, and interactivity for learning and practice purposes.

<Key Features

    Responsive layout for desktop, tablet and mobile

    Navigation menu with linked pages

    "Home","About Us", "Contact Us", "Our Menu" and "Our GAllery" sections

    Font Awesome icons integration

    Basic CSS styling

    Basic Java Script

<How to Run>

    Download or clone this repository.

    Open the index.html file in any modern browser.

    No server setup is required (static website).

<Dependencies>

    Font Awesome
    – for icons

    Google Fonts
    – for typography

<Usage Instructions>

    Edit the index.html file to update homepage content.

    Modified css/styles.css to change colors, fonts, or layout.

    Add new pages and link them through the navigation bar.
